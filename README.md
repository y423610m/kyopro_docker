# What is this?
docker file to create atcoder env for c++23(gcc)

# How to use

- clone
```
https://github.com/y423610m/kyopro_docker.git
```

- setup
```
cd kyopro_docker
source setup.bash
```

- build docker image (only first time)
```
kyopro_build_docker
```

- run, exec, commit docker
```
kyopro_run_docker
kyopro_exec_docker
kyopro_commit_docker
```
